Por que o modo ECB não é recomendado (manual de operação):

1) Use o programa cifra_imagem.py com o seguinte comando para gerar uma imagem cifrada em AES modo ECB:
    python cifra_imagem_AES_ECB_CBC.py test.jpg test1.jpg ECB

2) Use o programa cifra_imagem.py com o seguinte comando para gerar uma imagem cifrada em AES modo CBC:
    python cifra_imagem_AES_ECB_CBC.py test.jpg test2.jpg CBC

3) Compare os dois resultados
